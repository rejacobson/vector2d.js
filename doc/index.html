Using Vector2d.js

Vector2d can be loaded through Require.js (AMD), CommonJS or using script tags.

Require.js

define(['vector2d'], function(Vector2d) { ... });

CommonJS

var Vector2d = require('vector2d').Vector2d;

Script Tag

<script src="vector2d.js"></script>

Utility Methods

toString .toString() not chainable Returns a string representation of the vector components. 

var a = Vector2d(1, 2);

a.toString();
// returns "[1, 2]"

toArray .toArray() not chainable Returns the vector components as an array. 

var a = Vector2d(1, 2);

a.toArray();
// returns [1, 2]

clone .clone() Copies and returns the vector as a new Vector2d instance. 

var a = Vector2d(1, 2);

var b = a.clone();
// b !== a
// b == [1, 2]

set .set(x, y).set([x, y]) Sets the x and y components of a vector. 

var a = Vector2d();

a.set(1, 2);
// a == [1, 2]

a.set([3, 4]);
// a == [3, 4]

Basic Vector Math

add .add(v).add([x, y]) Adds one vector to another. 

var a = Vector2d(3, 4);
var b = Vector2d(1, 2);

a.add(b);
// a == [4, 6]

subtract .subtract(v).subtract([x, y]) Subtact a vector from another vector. 

var a = Vector2d(3, 4);
var b = Vector2d(1, 2);

a.subtract(b);
// a == [2, 2]

multiply .multiply(v).multiply([x, y]) Multiply a vector by another vector. 

var a = Vector2d(3, 4);
var b = Vector2d(1, 2);

a.multiply(b);
// a == [3, 8]

divide .divide(v).divide([x, y]) Divide a vector by another vector. 

var a = Vector2d(3, 4);
var b = Vector2d(1, 2);

a.divide(b);
// a == [3, 2]

Vector Manipulation

scale .scale(number) Multiplies the x and y components by the scalar argument. Useful for scaling a vector up or down. 

var a = Vector2d(2, 2);

a.scale(4);
// a == [8, 8]

a.scale(0.5)
// a == [4, 4]

truncate .truncate(max_length) Resizes the vector so it's length is no longer than `max_length`. 

var a = Vector2d(6, 0);

a.truncate(4);
// a == [4, 0]
// a.lengthOf() == 4

var a = Vector2d(6, 8);

a.truncate(5);
// a == [3, 4]
// a.lengthOf() == 5

reverse .reverse() Reverse the vector so that it points in the opposite direction. 

var a = Vector2d(1, 2);

a.reverse();
// a == [-1, -2]

rotate .rotate(angle, [origin]) Rotate a vector by the specified angle (in radians). 

var a = Vector2d(4, 0),
    angle_r = (90)*(0.0174532925);  // 90 degrees, to radians

a.rotate(angle_r);
// a == [0, 4]

If an origin is passed in, the vector rotates around the origin.

var a = Vector2d(6, 2),
    o = [2, 2],
    angle_r = (90)*(0.0174532925);  // 90 degrees, to radians

a.rotate(angle_r, o);
// a == [2, 6]

reflect .reflect(vector) Reflect a vector against another vector. For example, when reflecting a velocity vector against a wall that an object has collided with. Use the normal of the wall to reflect the velocity vector against. 

var a = Vector2d(4, 4),  // velocity
    n = [1, 0];          // wall normal. The wall is vertical, with a 0 x component.

a.reflect(n);
// a == [-4, 4]

unit .unit() Scales the vector to have a length of 1. 

var a = Vector2d(3, 4);

a.unit();
// returns [0.6, 0.8]

normal .normal() Rotates a vector by 90 degrees. If the y-axis points up then the resultant vector will be rotated to the right, otherwise it will be rotated to the left. 

var a = Vector2d(3, 4);

a.unit();
// returns [0.6, 0.8]

Queries

All vector queries are non-destructive

distanceTo .distanceTo(vector) not chainable Returns the scalar distance between one vector and another. Most useful when the vectors are point vectors. 

var a = Vector2d(3, 4);

a.distanceTo([-3, -4]);
// returns 10

distanceToSq .distanceTo(vector) not chainable Returns the squared scalar distance between one vector and another. 

var a = Vector2d(3, 4);

a.distanceTo([-3, -4]);
// returns 100

lengthOf .lengthOf() not chainable Returns the length of a vector. 

var a = Vector2d(3, 4);

a.lengthOf();
// returns 5

lengthOfSq .lengthOfSq() not chainable Returns the squared length of a vector. 

var a = Vector2d(3, 4);

a.lengthOfSq();
// returns 25

dot .dot(vector) not chainable Returns the dot product between two vectors. Dot product on wikipedia. Dot product on betterexplained.com 

var a = Vector2d(3, 4);

a.dot([1, 0]);
// returns 3

a.dot([0, 1]);
// return 4

cross .cross(vector) not chainable Returns the cross product between two vectors. 

var a = Vector2d(3, 4);

a.cross([1, 0]);
// returns -4

a.cross([0, 1]);
// return 3

angle .angle([vector]) not chainable Returns the angle, in radians, of a vector. 

var a = Vector2d(4, 4);

a.angle();
// returns 0.7853981633974483 (45 degrees)

When passed a vector parameter, it calculates the difference in angles between the two vectors.

var a = Vector2d(4, 4);

a.angle([4, 0]);
// returns -0.7853981633974483 (-45 degrees)
